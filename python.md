# python  第三方库的安装

## 安装方法
1. 找到文件所在位置
2. 选择IDLE
3. 右击打开文件所在位置
4. 找到并打开Scripts文件
5. 按住shift键在空白处右击
6. 点击“在此处打开命令窗口”
7. 输入“pip install 库名”
    (pip install requests)
8. 或者输入“pip install 库名 -i 镜像地址链接”
    (pip install jieba -i http://pypi.tuna.tsinghua.edu.cn/simple)
